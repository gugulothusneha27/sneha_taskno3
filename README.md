# ATM Interface

## Description

The ATM Interface is a Java console-based application that simulates the basic functions of an ATM. It allows users to log in, check their account balance, deposit money, withdraw money, and view transaction details through a simple menu-driven interface.

## Features

- User Login
- Check Account Balance
- Deposit Money
- Withdraw Money
- Transaction History
- Console-based Application

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- VS Code

## How to Run

1. Open the project in VS Code.
2. Compile the program:
   
   javac ATMInterface.java
   
3. Run the program:
   
   java ATMInterface
